# PoTminiEvo
This is the repository containing some scripts used in the following manuscript:  
Cruppe et al, 2026, Evolution of supernumerary chromosomes in wheat blast fungal pathogens

## content
Below briefly explains each analysis:  
01_tree: analysis from reads to a maximum likelihood tree    
02_asm: scripts for genome assembly and polishing  
03_annotation: genome annotation scripts  
04_miniC: [miniC](https://doi.org/10.1093/nargab/lqae108) analysis to infer core- or mini-like sequences  
05_chrcomp: chromosome comparisons using **gcomp** in [chrchomp](https://github.com/liu3zhenlab/chrcomp.git)  
06_chrstack: chromosome stacking using **homostack** in [homotools](https://github.com/liu3zhenlab/homotools.git)  
07_effector: mapping known effectors to genomes using [protmap](https://github.com/liu3zhenlab/protmap.git)  
08_clustering: syntenic sequence clusters to identify non-founder genomic regions
